#define COAP_OBS_MAX_FAIL   3

Number of confirmable notifications that may fail (i.e. time out without being ACKed) before an observer is removed. The maximum value for COAP_OBS_MAX_FAIL is 3.

#define COAP_OBS_MAX_NON   5

Number of notifications that may be sent non-confirmable before a confirmable message is sent to detect if observers are alive. The maximum allowed value here is 15.
